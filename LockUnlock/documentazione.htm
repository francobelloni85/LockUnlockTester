LockUnlock libreria
Libreria per disabilita alcune azioni all'utente.


USB storage

Disabilita globalmente le usb storage

  • GetUSBDevices()
  • ReadUSBStatusInRegistry()
  • SetUSBbyRegistry()

Permette di disabilitare le chiavette USB / hard disk esterni. Funziona sui dispositivi futuri. Se infatti una chiavetta è già inserita questa funzione non verrà disabilitata.


USB API

Disabilita la singola usb scrivendo nel registro di windows

Permette di disabilitare la singola usb scrivendo nel registro di windows.

Permette di disabilitare tutte le usb indistintamente, rendendo initulizzabile il pc


Devcon.exe

è un tool di microsoft che permette di disabilitare il singolo componente USB.

Questo tool è chiamato con la power shell e una volta eseguito un comando scrive su file il risultato. (il file è nella stessa cartella del programma)

La libreria permette di controllare e recuperare le periferiche collegate. Ci sono 2 funzioni:

GetDevConUSBList: Recupera solo le usb

GetDevConaAllHardware: recupera tutti i dispositivi del pc

è possibile disattivare il mouse e le altre periferiche. Non è possibile disattivare la tastiera con il devcon.

Per disattivare una usb si recupera la lista e poi si chiama la funzione EnableDeviceDevCon(value)


Virtual keyboard

Ci sono 3 metodi per disattivare la Virtual keyboard. Il primo e secondo sono tramite registro (ma non sembrano funzionare) il terzo è tramite gli eventi di windows.

Quando questa funzione viene chiamata ci si registra all'evento "NuovoProgramma". Quando un nuovo programma di windows viene eseguito si scatena questo evento. Se il nuovo programma è osk.exe ovvero la tastiera questo viene "terminato".


TaskManager

Ci sono 2 metodi per disattivare la il task manager. Il primo è tramite registro (ma non sembra funzionare) il secondo è tramite shell

Quando questa funzione viene chiamato il terminale che esegue il comando.


TaskBar

La task bar viene nascosta ma è lo stesso raggiungibile tramite il punsante sulla statiera di windows. Bisogna usare il keylogger per impedire che funzioni


keylogger

Viene usato per impedire all'utente di usare la tastiera oppure per disabilitare certi tasti/combinazione di tasti.

Il key logger si registra agli enventi che provengono dalla tastiera.

Se voglio disabilitare la tastiera è il key logger nella funzione private IntPtr LowLevelKeyboardProc(int nCode, UIntPtr wParam, IntPtr lParam) impedisce all'evento 'tasto premuto' di propagarsi

Si possono customizzare i tasti da disabilitare settando la variabile HashSet KeysToAvoid

Si può volendo collegare una callback all'evento per poter ad esempio stampare sulla console il tasto premuto
